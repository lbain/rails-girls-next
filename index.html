Welcome to Rails Girls Next!

We are very excited to have you at Rails Girls Next. Hopefully you'll come away with a better understanding of Ruby and how the internet works!

Schedule

Registration and coffee (9:30 - 9:45)

During the morning we'll have some coffee, thanks to Ruby Conf AU, so that we're all set for a great day of Rails Girls!

Welcome (9:45 - 10:00)

Outline of the day - since you have Ruby installed already, we can jump right into it.

We've broken the day down into 3 exercises so you can practice a few different things. There's no particular order, the list below is just a possible outline.

Coding with Ruby (10:00 - 11:30)

At Rails Girls you got to play with Ruby a bit, but now it's time to really understand some of its features and syntax.

Fun with Sinatra (11:45 - 1:00)

To give you a better foundation about how the web works, you'll build a simple website using Sinatra. It will be more hands-on than your original Rails Girls site, so be ready to write some code!

Lunch (1:00 - 2:00)

Delicious food from Ruby Conf AU.

Test Driven Development (2:00 - 3:20)

Learn a new way to code (and avoid errors!) by writing tests as you go.

Feedback Form (3:20 - 3:30)

We've got a very quick feedback form for you. This is the first time we're running Rails Girls Next, and we'd really apprecate your feedback.

Retro (3:30 - 4:00)

We'll go over everything we did for the day, and hear about your experiences from the day.
